### 2. 修复上传时间显示 ✅

**问题根源：**
- Nexus API字段名：`lastBlobUpdated`（不是 `lastModified`）
- 格式：ISO 8601字符串（不是long时间戳）
- 示例：`"2026-01-28T05:04:20.555Z"`

**修改的文件：**
- `NexusApiService.kt`

**新增方法：**
```kotlin
private fun parseIso8601ToTimestamp(dateStr: String): Long {
    // 支持两种格式：
    // 1. yyyy-MM-dd'T'HH:mm:ss.SSS'Z' (带毫秒)
    // 2. yyyy-MM-dd'T'HH:mm:ss'Z' (不带毫秒)
}
```

**效果对比：**
```
修复前：上传时间显示 "-"
修复后：上传时间显示 "2026-01-28 13:04:20"
```

### 3. 首次打开自动加载 ✅

**问题：**
- 打开工具窗口是空白的
- 需要手动点击刷新按钮

**修改的文件：**
- `NexusVersionToolWindow.kt`

**实现逻辑：**
```kotlin
SwingUtilities.invokeLater {
    Thread {
        Thread.sleep(1000)  // 延迟1秒，确保Maven项目加载
        ApplicationManager.getApplication().invokeLater {
            refreshAllApiModules()  // 自动刷新
        }
    }.start()
}
```

**用户体验：**
```
旧版流程：
1. 打开工具窗口
2. 看到空白界面
3. 手动点击"刷新所有API模块"
4. 等待加载
5. 看到数据

新版流程：
1. 打开工具窗口
2. 自动开始加载
3. 看到数据
```
